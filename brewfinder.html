BREWFINDER

An Android application for finding breweries

DESCRIPTION

Brewfinder is an Android application that allows users to search for breweries in the United States. The application is built using the Google Maps API and the BreweryDB API.

TECHNOLOGIES

  • Kotlin
  • Android Studio
  • Google Maps API
  • BreweryDB API
